# Golden files for integration tests

## fdn_reverb_impulse.bin

Little-endian f32 samples. Generated by:

    cargo test -p patches-integration-tests -- --ignored generate_fdn_reverb_golden_file

Parameters used:
- Sample rate: 48000 Hz
- size: 0.5, pre_delay: 0.0, decay: 0.5, brightness: 0.5, modulation: 0.0, character: "plate", dry_wet: 1.0
- Input: unit impulse (1.0 at sample 0, 0.0 thereafter)
- Length: 2048 samples (captures early reflections)
- Outputs: left then right channel interleaved (4096 f32 values total)

Regenerate only when the FDN reverb algorithm is intentionally changed.
